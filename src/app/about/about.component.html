📖 About This Project

🌍 Project Overview

This project is a **joint website development** focused on learning and implementing **Angular Routing**. We created a structured and dynamic website using **Angular, Bootstrap, and TypeScript**.

🛠️ Technologies Used

  • Angular (Routing & Components)
  • Bootstrap for Styling
  • TypeScript
  • HTML & CSS

📚 What We Learned

Throughout this project, we **gained hands-on experience** in: - Implementing **Angular Routing** for navigation - Using **Bootstrap** for modern UI design - Structuring **components** efficiently - Managing **dynamic content and router links**

🚀 Development Process:

1️⃣ **Planning** - Defined the website structure and routing setup. 2️⃣ **Development** - Implemented Angular components & Bootstrap styles. 3️⃣ **Testing & Debugging** - Fixed navigation and layout issues. 4️⃣ **Deployment** - Finalized and deployed the project.
